## ADDED Requirements

### Requirement: 当前输入显示
系统 SHALL 在游戏区域显示用户当前的输入内容，让用户知道正在输入什么。

#### Scenario: 输入实时显示
- **WHEN** 用户按下键盘
- **THEN** 系统在固定位置显示当前输入的字符

#### Scenario: 空格/回车清空输入
- **WHEN** 用户按下空格键或回车键
- **THEN** 系统清空当前输入显示

#### Scenario: 超限清空输入
- **WHEN** 当前输入长度达到 4 个字符且未匹配成功
- **THEN** 系统自动清空当前输入

#### Scenario: 输入清空
- **WHEN** 正确输入匹配当前汉字
- **THEN** 系统清空当前输入显示

### Requirement: 正确历史记录
系统 SHALL 记录和展示用户在本局游戏中所有正确输入的汉字。

#### Scenario: 正确汉字记录
- **WHEN** 用户正确输入汉字
- **THEN** 系统将该汉字添加到正确历史列表

#### Scenario: 历史记录显示
- **WHEN** 游戏进行中
- **THEN** 系统在游戏区域下方显示已正确输入的所有汉字，按输入顺序排列

#### Scenario: 历史记录滚动
- **WHEN** 正确汉字数量超过显示区域
- **THEN** 系统提供滚动条查看完整历史

#### Scenario: 游戏结束清空
- **WHEN** 游戏结束或重新开始
- **THEN** 系统清空正确历史列表
