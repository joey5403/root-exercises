# 五笔 98 字根练习工具

一个交互式的五笔 98 字根练习工具，帮助你快速记忆字根键位。

## 🎯 功能特点

- **随机练习**: 随机显示字根图片，测试你的键位记忆
- **实时反馈**: 立即显示答案正确/错误，加深印象
- **错题复习**: 自动记录错误，支持专项复习错题
- **学习统计**: 追踪正确率、错误分布、掌握程度
- **本地存储**: 数据保存在浏览器，关闭不丢失
- **响应式设计**: 支持电脑和移动端使用
- **589 个字根**: 100% 覆盖所有字根图片（键名字、横区、竖区、撇区、捺区、折区、全部字根、一级简码、繁体字根）
- **分类练习**: 支持按区域筛选字根，针对性练习

## 🚀 使用方法

### 方式一：直接打开
双击 `index.html` 文件，在浏览器中打开即可使用。

### 方式二：本地服务器
```bash
# 使用 Python
cd wubi98-practice
python -m http.server 8000

# 使用 Node.js
npx serve wubi98-practice

# 然后访问 http://localhost:8000
```

## 📖 使用说明

1. **选择分类**: 从顶部分类下拉框选择要练习的字根区域（可选：全部、键名字、横区、竖区、撇区、捺区、折区、全部字根、一级简码、繁体字根）
2. **开始练习**: 点击"开始练习"按钮
3. **输入答案**: 根据显示的字根图片，按下对应的键位字母
4. **查看反馈**: 
   - ✓ 正确：绿色提示，自动进入下一题
   - ✗ 错误：红色提示，显示正确答案
5. **切换模式**:
   - 全部练习：随机显示所有字根
   - 错题复习：只显示曾经答错的字根
6. **查看统计**: 点击"统计"按钮查看详细数据

## 📊 统计数据说明

- **总练习次数**: 累计练习的题目数量
- **正确率**: 答对题目数 / 总练习数
- **已掌握**: 错误次数为 0 的字根数量
- **需复习**: 有错误记录的字根数量
- **Top 5 错误字根**: 错误次数最多的 5 个字根

## 🔧 自定义配置

### 添加更多字根

编辑 `data.js` 文件，在 `radicals` 数组中添加新字根：

```javascript
{ key: 'g', image: '字根/新文件夹/image.jpg', name: '字根名', category: '分类名' }
```

### 修改图片路径

如果图片文件位置有变化，需要更新 `data.js` 中的 `image` 路径。

## 📝 数据存储

学习数据保存在浏览器的 `localStorage` 中，键名为 `wubi98_stats`。

数据结构：
```javascript
{
  "g": { errors: 3, correct: 10, lastPracticed: timestamp },
  "f": { errors: 5, correct: 8, lastPracticed: timestamp },
  // ...
}
```

### 清除数据

点击统计面板底部的"重置统计数据"按钮，或手动清除浏览器缓存。

## 💡 学习建议

1. **循序渐进**: 先练习键名字，再扩展到其他字根
2. **错题重点**: 使用"错题复习"模式，强化薄弱环节
3. **定时练习**: 每天练习 10-15 分钟，形成肌肉记忆
4. **结合实战**: 配合五笔打字练习，巩固记忆
